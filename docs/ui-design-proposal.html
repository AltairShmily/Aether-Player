Part 01

Jellyfin Media Player 界面分析

基于 Terminus-Media/jellyfin-media-player 项目的完整界面拆解与评估

架构

Qt + WebEngine 壳

外壳是 Qt5 原生窗口 (KonvergoWindow)，内部嵌入 Chromium 渲染 jellyfin-web-jmp，播放器使用 MPV 通过 QWebChannel 与 Web 层通信。

视觉语言

标准暗色主题

背景 #101010，强调色 #00a4dc (Jellyfin 蓝)，卡片无边框，OSD 使用渐变遮罩 + 毛玻璃。整体功能导向，缺乏品牌个性。

交互模式

TV / Desktop 双模式

TV 模式: 大卡片网格 + 方向键导航。Desktop 模式: 顶部导航栏 + 鼠标操作。OSD 300ms 淡入淡出。

评估

功能完善，美学不足

✅ 暗色沉浸 · ✅ OSD 渐变自然 · ✅ 卡片分类清晰
❌ 品牌辨识度弱 · ❌ 无动态特效 · ❌ 颜色系统单一

Part 05

JMP vs Aether 对比

从标准工具到沉浸体验的跨越

维度 Jellyfin Media Player Aether Player (新)
背景 #101010 纯色 #0A0E14 深空渐变 + 星空粒子
主色 #00a4dc 单一蓝 #00D4FF 星辰青 + 5色系
卡片 扁平无特效 毛玻璃 + 悬停发光
动效 基础 CSS transition 弹性曲线 + 交错动画
进度条 单色直线 渐变 + hover 扩展 + 发光
字体 系统默认 Inter + JetBrains Mono
品牌感 弱 (Web 客户端外观) 强 (星系主题完整体系)

  • 登录/服务器选择页
Phase 4 · Week 4

🎬 播放器 & 动效

打造专业级播放器界面，完善所有微交互
